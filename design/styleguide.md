

# Component driven module development
## Use BEM class names for better modularity.

```
.site-search {} /* Block */ 
.site-search__field {} /* Element */
.site-search--full {} /* Modifier */
```
### Examples
```
.page { }

.content { }
  .contentcontent_header {}

.sub-content { }

.footer { }

  .footer__copyright { }
```

#### Modifers 

```
.button { }
.button--green { }
```
